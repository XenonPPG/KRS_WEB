# KRS WEB
### Фронтенд системы KRS, который взаимодействует с [KRS_JWT](https://github.com/XenonPPG/KRS_JWT)
### Использует Vue3 и библиотеку компонентов shadcn

---

# Описание системы

**KRS** - учебный проект, созданный для исследования микросервисной архитектуры и аутентификации через JWT
